# VgcCollege

ASP.NET Core MVC application for managing branches, courses, students, faculty, enrolments, attendance, assignments, and exams.

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- ASP.NET Core Identity
- xUnit
- GitHub Actions

## How to Run
1. Open the solution in Visual Studio
2. Run:
   Update-Database
3. Press F5
